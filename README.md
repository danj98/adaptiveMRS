# adaptiveMRS
Contains the master's project for an adaptive multirobot system.

## Prerequisites

- C++11 compiler
- CMake
- YAML-cpp library

## Installation

1. Clone the repository.
2. Install the dependencies.
3. Build the project using CMake:
```bash
mkdir build #if not already created
cd build
cmake ..
make
```

## Usage

```bash
cd build
./adaptiveMRS <path_to_config_file>
```

## Configuration

The configuration file is a YAML file which can be found in the input folder.
